ScatterScam

Détection de contrefaçons d'applications mobiles.

ScatterScam est une plateforme de détection et d'identification d'applications Android contrefaites.

Qu'est-ce qu'une application contrefaite ?

C'est une application qui, la plupart du temps, agit exactement comme l'application officielle : elle offre les mêmes services, les mêmes fonctionnalités; elle est visuellement identique; mais elle a été modifiée pour intégrer du code malveillant.

Ce code malveillant peut, par exemple : provoquer des fuites de données (dont des données sensibles comme les identifiants de connexion), abuser des permissions accordées à votre application pour en faire un usage illégitime.

Les principaux leviers recherchés par les attaquants pour choisir votre application comme vecteur d'attaque sont la plupart du temps : votre image de marque, votre notoriété, la confiance que vous offrent vos clients.

Comment fonctionne ScatterScam ?

ScatterScam s'appuie sur plusieurs robots parcourant quotidiennement Internet à la recherche de nouvelles applications publiées.

Le moteur de signatures intégré à ScatterScam permet, en temps réel, la détection de contrefaçons, le suivi et l'identification des applications légitimes contrefaites.

Lorsque ScatterScam détecte une contrefaçon, il vous en avertit immédiatement et vous permet de prendre connaissance des détails récoltés sur cette contrefaçon.

Fonctionnalités
Monitoring & Alertes

Recevez une alerte lorsque ScatterScam identifie une version contrefaite de votre application et suivez en temps réel sa propagation.

Exemple d’usage : vous éditez une application sensible en terme de sécurité (banque, messagerie, etc.) et souhaitez réagir au plus vite dès l'apparition d'une contrefaçon afin de protéger vos utilisateurs.

SDK & Vérification

Intégrez le SDK ScatterScam dans votre application pour vous assurer qu'une fois installée elle n'interagit qu'avec des applications légitimes.

Exemple d’usage : vous éditez un gestionnaire de mots de passe sur mobile et souhaitez vous assurer de ne jamais communiquer les identifiants de vos utilisateurs à une application contrefaite.
